# 好創短影音腳本生成器 (Reels/TikTok/Shorts)

此技能專為好創整合行銷設計，用於生成高轉換率、符合台灣市場語境的短影音腳本。
執行此技能前，必須同步載入 `haochuang-brand-context` 與 `marketing-ai-workflow` 技能。

## 核心產出原則（嚴格遵守）

1. **成效至上**：所有台詞、畫面、音效設計皆服務於「導購轉換率」或「核心行銷目標」，拒絕無效的炫技。
2. **台灣市場網感**：必須深度結合台灣時下語氣、幽默感或諧音梗，避免生硬的翻譯腔或非在地用語。
3. **拍攝極簡、質感頂流**：腳本設計需考量執行成本（基於現有 4 人團隊產能：陳涵勻/黃子權/阿韋/小鑫），畫面與轉場設計要簡單易拍，但透過節奏與內容達到 META 投廣的頂級質感。
4. **反方攻擊驗證**：在產出最終腳本前，必須先以「最嚴厲的反方角度」自我審視，找出邏輯漏洞、跳出率高的風險點，並進行優化。

## 使用場景與邏輯分支

收到短影音腳本需求時，先確認以下資訊（若資訊不足，立即中斷並提問）：

1. **用途歸屬**：
   - **好創自有品牌**：語氣第一人稱「好創」，強調專業、直接、解決痛點，存入 `haochuang-content` repo。
   - **客戶代操**：確認客戶品牌語氣、產業限制，存入客戶專屬 repo。
2. **主力平台**：Reels / TikTok / YouTube Shorts（影響前 3 秒鉤子節奏與整體時長）。
3. **行銷目標**：直接導購、名單收集、品牌曝光。

## 腳本生成工作流

### 步驟一：核心架構設定
根據需求，設定影片的 4 個核心節點：
1. **黃金 3 秒鉤子 (Hook)**：痛點直擊 / 視覺衝擊 / 懸念反轉。
2. **痛點共鳴 (Pain Point)**：放大問題，讓受眾覺得「這就是在說我」。
3. **解法與信任 (Solution & Trust)**：提出解法（產品/服務），並附帶一個強而有力的社會證明或數據。
4. **行動呼籲 (CTA)**：明確、單一的指令（如：點擊主頁連結、留言領取）。

### 步驟二：自我審視與優化（反方攻擊）
在撰寫完整腳本前，自我提問：
- 前 3 秒夠不夠抓眼球？會不會被滑掉？
- 台灣受眾聽得懂這個梗或痛點嗎？
- 畫面設計會不會太複雜，超出阿韋的剪輯負荷或拍攝成本？
- CTA 是否夠直接？

### 步驟三：雙格式輸出交付
腳本必須同時提供以下兩種格式，讓子權與阿韋都能高效使用：

#### 格式一：純文字分鏡（供快速閱讀與確認口播稿）
```markdown
【標題】（吸引人的標題）
【時長預估】（例如：30-45秒）

[00:00-00:03] 鉤子：(畫面描述) + (口播台詞)
[00:03-00:15] 痛點：(畫面描述) + (口播台詞)
[00:15-00:25] 解法：(畫面描述) + (口播台詞)
[00:25-00:30] CTA：(畫面描述) + (口播台詞)
```

#### 格式二：結構化表格（供阿韋剪輯與拍攝執行）
| 鏡號 | 時間軸 | 畫面設計 (視覺/動作) | 口播台詞 (字幕) | 音效/音樂/特效指示 |
|------|--------|----------------------|-----------------|--------------------|
| 1    | 0-3s   |                      |                 |                    |
| 2    | 3-10s  |                      |                 |                    |
| ...  | ...    |                      |                 |                    |
