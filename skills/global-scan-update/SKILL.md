# 全局掃描與更新 (Global Scan & Update)

此技能供好創團隊成員使用，負責執行**最高成本、最全面**的系統現狀調查，並產出唯一的「單一真實來源」快照文件（全局知識地圖）。

## 🚨 觸發時機與限制

- **唯一觸發條件**：只有當使用者**明確要求**「全面掃描更新」、「更新全局知識地圖」或「重新掃描所有系統」時才可觸發。
- **嚴格禁止**：禁止在日常對話、一般任務或未經授權的情況下自動觸發此技能，因為全量掃描極其耗時且消耗大量 Token。

## ⚙️ 核心工作流 (Core Workflow)

### 步驟一：執行全量掃描 (Full System Reconnaissance)

必須依序掃描以下五大系統，不遺漏任何最新狀態：

1. **Monday.com (專案與業務中樞)**
   - 掃描所有活躍看板的欄位結構與狀態分佈。
   - 提取最新報價單、合約流程與專案執行現狀。
2. **Google Drive (檔案與素材庫)**
   - 掃描核心資料夾的變動。
   - 讀取最新的財務或報價試算表。
3. **Notion (知識與 SOP 庫房)**
   - 掃描「好創庫房」的最新架構與內容變動。
   - 讀取各項 SOP 與操作手冊的最新版本。
4. **GitHub (程式碼與自動化庫)**
   - 掃描 `meta-script-generator`、`viral-factory` 等核心專案的最新 Commit 與架構變更。
5. **Slack (溝通與通知節點)**
   - 確認最新的頻道結構與自動化 Webhook 綁定狀態。

### 步驟二：產出《全局知識地圖》Markdown

將掃描結果統整為一份結構化的 Markdown 文件，必須包含以下核心區塊：
- **公司定位與營收結構**（A線＝短影音代操與投放／B線＝網站短影音廣告整包月費案／C線＝老闆個人 IP 經營／D線＝外部合作夥伴代理，各線現況與定價以公司大腦為準）
- **團隊架構與產能防禦機制**（現有 4 人：CEO 陳涵勻、COO 黃子權、剪輯師阿韋、企劃小鑫；招募中：剪輯師、行政、專案管理、短影音操盤手、AI 影音導演）
- **系統工具棧與自動化現狀**（Monday、Slack、Notion、GitHub 的資料流向）
- **最新發現的業務缺口或流程斷點**
- **更新時間戳記**（精確到分鐘）

### 步驟三：執行三備份同步 (Triple Backup Sync)

將產出的 Markdown 文件同步寫入以下三個位置，並確保**舊版與新版並存**（以日期後綴區分，如 `global_map_v2026.06.14.md`）：

1. **Google Drive**：上傳至全局知識地圖專屬資料夾。
   - 參考舊檔 ID: `1y7M7NCQRhVZHuFIeS9d4yxEvoHUZV37N`
2. **Notion**：更新至「好創庫房」的全局知識地圖頁面。
   - 參考頁面 ID: `37b97a06-fae5-817f-8028-ed2eadea5bcb`
3. **GitHub**：Commit 並 Push 至好創的核心 Repository（如 `jayw19911125-sys/WEB` 或系統配置庫）的根目錄。

### 步驟四：交付報告

向團隊回報更新完成，並提供三個備份位置的連結與本次掃描發現的重大變更摘要。
