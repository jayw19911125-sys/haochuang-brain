# 好創整合行銷｜AI 統一知識來源

這個 repo 是好創整合行銷的「公司大腦」，目的是讓所有 AI 工具（Manus、ChatGPT、Claude、Gemini…）讀到**同一份、永遠最新**的公司資料。

## 主要檔案

- `haochuang_brain.md`：公司大腦完整內容（唯一真實來源，要改就改這份）。
- `llms.txt`：給 AI 的索引檔。
- `README.md`：本說明。

## 唯一來源網址（複製這條給 AI）

公司大腦 raw 網址（永遠最新）：

```
https://raw.githubusercontent.com/jayw19911125-sys/haochuang-brain/master/haochuang_brain.md
```

## 維護規則

1. 公司現況有變動 → 只改 `haochuang_brain.md`，commit 後網址內容立刻更新。
2. 各 AI 平台用「自訂指令」指向上面那條網址，叫它回答前先去讀。
3. 不要在各平台分別維護內容，避免再次資訊脫節。
